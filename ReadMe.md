# 🏥 Hospital Appointment Management System

A simple C-based project to manage doctors, patients, and appointments using a menu-driven approach.

---

## 📌 Features

* Add & view doctors
* Add & view patients
* Book & view appointments
* Basic login system

---

## 🧠 How it works

* Uses structures and arrays to store data
* Data is temporary (resets after program ends)
* Login required before accessing the system

---
